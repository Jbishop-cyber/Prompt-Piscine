# Overview:

## What is a prompt?

A prompt is a set of questions or instructions given to AI to get a response.

## Anatomy of a prompt (system, user, assistant messages)

The anatomy of a prompt is divided into three:

**System:** - This sets the tone and the behaviour of the AI

**User:** - This is the message you send to AI. It is what you ask or instruct AI to do.

**Assistant Messages:** - This is the AI's response to what you asked.

## Zero-shot vs few-shot prompting

- **Zero-shot prompting:** This is when you ask AI to perform a task without giving it any examples.

- **Few-shot prompting:** This is when you give AI a few examples and ask it to conclude or use the pattern.

## Temperature and Top-p

- Pararmeters like temperature and top-p controls the creativity and structure of the responses.
