## Overview

This quest focuses on:

**How to detect and fix misleading or inaccurate outputs:** 

Learning how to identify hallucinations, vague and fabricated facts inside the AI output and also applying strategies (such as properly rewriting the prompt) to correct them.
  
**Anchoring responses to data, constraints, or reference material:**

Guiding the model to work with only the provided texts, facts and well structured inputs to produce accurate outputs.

## Learning Objectives

By completing this quest, I should be able to:

**Spot vague or fabricated information in outputs:**

To recognize responses that are false, inaccurate and fabricated. Most times this can easily be done by asking the model to reference the source.

**Rewrite prompts to add clear constraints:**

To rewrite prompts that would produce accurate outputs, that can be easily verified.

**Use delimiters and formatting to keep responses on topic:**

To make use of delimiters that can restrict the model's focus to a specific and important text, in order to improve the accuracy of the responses. 

```
Note:

A well written context can help AI to prevent generating misleading and inaccurate information.
```
