# Overview

The aim of this quest is to explore how generative prompts can be used for creative output. It also explores how text can be guided to produce stories, visuals and marketing contents through a well structured prompt. It also showed how we can apply prompting for real world cases such as UX, resume writing, ideation and designs.

# Learning Objectives

By completing this quest, I should be able to:

**Write prompts for creative generation tasks:**

To write design prompts that can efficiently guide models in producing a well structured output such as short stories. 

**Personalize prompts for industry-specific applications:**

To adjust prompts based on the role, the context and the required skills needed in order to produce professional outputs like resume summaries.

**Explore prompts for both text-based and image-based models:**

To understand just how been specific and descriptive can influence the quality of the output across various generative models.

```
Note:

For a model to produce meaningful and controlled outputs, the prompts has to be clear, intentional and descriptive.
```
