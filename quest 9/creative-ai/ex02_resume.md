# Objective

## Apply prompting to professional use cases.

### Step 1: Write a prompt that generates a resume summary based on role and skills

```
Prompt:

Create a four-sentence professional summary for a software engineer skilled in python development, cloud systems and backend design. Focus on technical problem-solving, and software reliability.
```
```
Output:

A versatile Software Engineer with extensive experience in Python development and the architectural design of high-availability back-end systems.

I specialize in solving complex technical challenges by engineering resilient, cloud-native infrastructures that ensure maximum up-time and seamless scalability.

My approach prioritizes software reliability, utilizing rigorous testing frameworks and automated monitoring to preemptively identify and resolve system bottlenecks.

I am dedicated to optimizing performance through clean, efficient code and distributed system design to support mission-critical business operations.
```

### Step 2: Adjust the prompt for a different role (Journalist)

```
Prompt:

Create a 4-sentence professional summary for a journalist who is skilled in investigative reporting, communication, and storytelling. Focus on their public information delivery and their clarity when creating content.
```
```
Output:

A dedicated Investigative Journalist committed to high-impact public information delivery and the pursuit of transparent, fact-based reporting.

I specialize in distilling complex data and sensitive investigations into clear, accessible storytelling that bridges the gap between intricate issues and community understanding.

By prioritizing content clarity and precise communication, I ensure that critical narratives remain engaging and easy to navigate for diverse audiences.

My professional approach focuses on maintaining the highest standards of accuracy to foster public trust and provide the clarity necessary for informed civic engagement.
```

### Step 3: Compare how the outputs change depending on the role

The output of each role changed based on the specification of the role. The engineering summary focuses more on it's technical skills, software expertise's and development abilities, while the journalist summary focuses more on it's storytelling and communication skills. This just shows that prompt personalization, if used correctly, can generate industry specific summaries.
